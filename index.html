Introduction

The aim of the project is to recommend books by determining degree of similarity between pairs of books. Given a book, the task is to find a book and classify the pair in one of the three classes:
Class 1: If the given pair is highly similar.
Class 0: If the given pair is moderately similar.
Class -1: If the given pair is not similar at all.
